# Typing_test


##  Description

This is a simple Python project to check typing speed and accuracy.

##  Used

Python, Tkinter, time, random



Run the file using:
python filename.py

##  Working

* Click Start
* Type the sentence
* Click Check
* Get WPM and Accuracy

##  Formula

WPM = (Words / Time) × 60
Accuracy = (Correct / Total) × 100

## Conclusion

This project helps to improve typing speed and learn Python GUI.
